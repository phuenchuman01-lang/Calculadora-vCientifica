# Calculadora-vCientifica
Simulacion de una calculadora como programa LDP java

### Integrantes y roles
- [Katherine Llanquinao Gallardo](https://github.com/Katherine23015) Responsable de la opción A (Operaciones aritméticas, mayor/menor, potencia y porcentaje).
  * Implementación de la lógica en `CalculadoraBasica.java`.
  * Creación de pruebas unitarias en `CalculadoraBasicaTest.java`.
- [Luis Arias Quezada](https://github.com/Lgarias11). Responsable de la Opción B (Manejo de excepciones y validación de datos).
  * Implementación de escudos matemáticos (división por cero y potencias indeterminadas) en `CalculadoraBasica.java`.
  * Creación del sistema de validación de formato de entrada por teclado en `EntradaDatos.java`.
  * Estructuración de pruebas de robustez para la consola usando bloques `try-catch` en `TestMain.java`.
- [Patricio Huenchuman Calful](https://github.com/phuenchuman01-lang) Responsable de la Opción C (Creacion de menu para las opciones de la calculadora).
  * Implementacion de Menu Interactivo `Main.java`
  * Agregado de clases para Ecuaciones cuadraticas y sus pruebas unitarias `EcuacionesCuadraticas.java` y `EcuacionesCuadraticasTest.java`
